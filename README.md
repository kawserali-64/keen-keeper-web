# 👥 KeenKeeper

## 🚀 Description

KeenKeeper is a simple and user-friendly web application designed to help users manage their friendships and track interactions. It allows users to stay connected by monitoring communication and maintaining meaningful relationships.

---

## 🛠️ Technologies Used

* Next.js (App Router)
* React.js
* Tailwind CSS
* DaisyUI
* Recharts
* React Toastify
* Lucide React Icons

---

## ✨ Key Features

### 🔹 Key Feature 1: Friend List & Details

Displays all friends in a responsive grid layout.

Each card includes:

* Profile picture
* Name
* Status
* Tags

Clicking on a friend opens a detailed profile page with more information.

---

### 🔹 Key Feature 2: Interaction Tracking

Users can quickly log interactions:

* 📞 Call
* 💬 Text
* 🎥 Video

Each interaction:

* Adds a new entry to the timeline
* Shows a toast notification

---

### 🔹 Key Feature 3: Timeline & Analytics

* Timeline page displays all interaction history
* Users can filter interactions by Call, Text, or Video
* Analytics page visualizes interaction data using a pie chart
